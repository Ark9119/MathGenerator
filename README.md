# Проект MathGenerator

## О проекте:
Экзаменационная утилита для проверки навыков вычипсления простых математических операций в формате тестового задания.
Разрабатывалась для оценки знаний школьников различных этапов обучения.

## Возможности:
Генерирует математические примеры с возможностью самостоятельно указывать параметры генерации:
- количество примеров
- количество чисел в примере
- выбор вида математической операции из умножения, деления, сложения, вычитания и любых их комбинаций в случайном порядке
- выбор диапазона чисел из которых будет составлен пример в случайном порядке
- выбор дополнительных опций которые можно включить или выключить при генерации:
    - отрицательные числа в ответах
    - дробные числа в ответах
    - ноль в ответе
- быстрая проверка результатов выполнения задания(реализована через технические файлы) с вычислением процента успешных ответов и возможностью просмотреть правильные ответы, сделав работу над ошибками.
- возможность повторно решить примеры
При генерации создаются и в последующем перезаписываются четыре файла в директории исполняемого файла MathGenerator.exe:
 - examples.txt - технический файл с примерами
 - answers.txt - технический файл с ответами
 - examples.docx - Microsoft Word файл с примерами для печати
 - answers.docx - Microsoft Word файл с ответами для печати

## Запуск:
1) Клонировать репозиторий/скачать архив с репозиторием
2) Запустить файл build.bat
3) Дождаться сборки приложения из исходников
4) В директории с файлом build.bat создастся директория dist в которой будет исполняемый файл MathGenerator.exe
5) Запустить MathGenerator.exe
Или через любой редактор кода(без создания исполняемого файла):
1) создать и активировать виртуальное окружение
```
python -m venv venv
```
```
source env/bin/activate
```
2) установить зависимости из файла requirements.txt:
```
pip install -r requirements.txt
```
3) запустить проект 
```
python interface_for_main.py
```
